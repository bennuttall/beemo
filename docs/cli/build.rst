===========
beemo build
===========

.. program:: beemo-build

Build the site.

.. code-block:: text

    Usage: beemo build [OPTIONS]

Options
=======

.. option:: --pages-dir PATH

    Pages content directory.

.. option:: --posts-dir PATH

    Posts content directory.

.. option:: --static-dir PATH

    Static files directory.

.. option:: --templates-dir PATH

    Chameleon templates directory.

.. option:: --output-dir PATH

    Output directory.

.. option:: --blog-root PATH

    URL path prefix for blog pages.

.. option:: --base-url TEXT

    Site base URL e.g. ``https://mysite.com``.

.. option:: --help

    Show this message and exit.
