==========
beemo logs
==========

.. program:: beemo-logs

Process Apache log gz files into CSV.

.. code-block:: text

    Usage: beemo logs [OPTIONS] [INPUT]

Arguments
=========

.. option:: INPUT

    gz file or directory of gz files.

Options
=======

.. option:: --csv-dir PATH

    Output directory for CSV files.

.. option:: --pattern TEXT

    Filename glob pattern when input is a directory.

.. option:: --help

    Show this message and exit.
